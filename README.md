# :rocket: SpaceX Launches || Demo App

> React SPA that uses GraphQL with the open [SpaceX Land API](https://api.spacex.land/graphql/) created by @swcarlosrj

---

## Tech Stack

- [React](https://github.com/facebook/create-react-app) - JS library, bootstrapped with create-react-app
- [Apollo Client](https://github.com/apollographql/apollo-client) - Fetch, cache, and modify GraphQL data
- [Zustand](https://github.com/react-spring/zustand) - Global state management
- [TailwindCSS](https://github.com/tailwindcss/tailwindcss) - Utility based styling
- [Framer Motion](https://github.com/framer/motion) - Animation library
